## Community Projects
Due to vllmAPI solely being an API server, the community has provided custom-tailored applications to communicate with a vllmAPI backend.

> [!NOTE]
> If you would like to showcase a community project, feel free to contact a dev!

Below is a list of projects that integrate vllmAPI:
- [vllmAPI Gradio Loader](https://github.com/lesj0610/vllmAPI-gradio-loader) by Doctor Shotgun - Gradio WebUI for loading/unloading models and loras via vllmAPI.
- [ST vllmAPI Loader](https://github.com/theroyallab/ST-tabbyapi-loader) by kingbri - SillyTavern extension for loading and unloading models via vllmAPI.
- [SillyTavern Launcher](https://github.com/SillyTavern/SillyTavern-Launcher) by deffcolony - CLI manager for AI tools which includes packaged setup and management of vllmAPI.
- [SillyTavern](https://github.com/SillyTavern/SillyTavern) by Cohee and RossAscends - An AI frontend for power users that fully supports vllmAPI's generation options.
- [Agnaistic](https://agnai.chat/) by Sceik - Multi-user AI role-play chat frontend that fully supports vllmAPI's generation options.
- [DS-LLM-WebUI](https://github.com/DocShotgun/ds-llm-webui) by Doctor Shotgun - A tool use assistant for local LLMs which is fully designed around vllmAPI.
- [PolyMind](https://github.com/itsme2417/PolyMind) by Itsme2417 - A multimodal function calling LLM webui that is designed to be used with vllmAPI.
